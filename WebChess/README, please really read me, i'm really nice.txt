Bonjour,
nous sommes heureux de vous présenter notre projet.
Si vous êtes arrivés jusque là, vous avez surement reçu l'archive de notre projet et vous êtes sûrement dans le dossier Webchess.
Pour lancer notre serveur, il suffit de lancer chess.jar.
Pour cela, il faut se placer dans un terminal dans le dossier Webchess (soit en ouvrant le terminal directement dans le dossier, soit grâce à une série de cd).
Une fois le terminal dans le bon dossier, il faut écrire "java -jar chess.jar" puis appuyer sur le bouton entrée (le joli bouton à droite des lettres avec une flèche qui va vers le bas puis vers la gauche).
Si tout se passe bien le serveur doit se lancer, sinon tant-pis... (non non on plaisante, si ça marche pas le code du projet est dans src et le main est dans Web.java)
Une fois le serveur lancé, ouvrir un navigateur (tant que c'est pas internet explorer ça nous va) et ecrire l'adresse "localhost:7777", et la page devrait s'afficher.
Une fois dans la page, le plateau de jeu est au milieu et les options sont dans le bandeau sur la droite (mais ça c'est écrit sur la page comme vous nous l'avez conseillé).
Pour quitter le serveur on aurait aimé vous faire un petit script qui tue le processus mais bon on a pas réussi alors il va falloir tuer le processus vous même ('ps -ef |grep chess.jar' et 'kill -9 XXXX' avec XXXX le numero du processus devraient vous être utile, mais bon ça vous devez le savoir).
Merci beaucoup et bon jeu (et comme le disait notre ami Gary Kasparov, "Il n'y a pas de sport plus violent que les échecs.")
